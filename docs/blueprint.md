# **App Name**: QAttend

## Core Features:

- Student Registration: Allow students to register with their academic details and verify their email.
- QR Code Generation: Generate persistent, time-bound, and class-specific QR codes for each class.
- Attendance Marking: Allow students to scan QR codes during scheduled class times to mark their attendance.
- Real-time Attendance Tracking: Display a real-time list of present students as they scan the QR code.
- Attendance History: View and export past attendance records.
- Anomaly Detection: Use an AI tool to flag suspicious attendance patterns (e.g., same device scanning for multiple students).
- User Role Management: Distinguish between student and teacher roles with distinct functionalities and access levels.

## Style Guidelines:

- Primary color: Light blue (#A7D1E8) to convey trust and reliability in the attendance tracking process.
- Background color: Very light blue (#F2F9FC), close in hue to the primary color.
- Accent color: Soft green (#B8E3A9) for positive confirmations, such as successful attendance marking.
- Font: 'PT Sans', a humanist sans-serif that is modern but with a little warmth; good for both headlines and body text.
- Use clear, professional icons for navigation and attendance status.
- Design a clean, intuitive layout for both student and teacher dashboards, focusing on easy navigation and access to key features.
- Incorporate subtle animations, such as a loading animation during QR code scanning or a celebratory animation upon successful attendance marking.
