# Mobile Camera Access Guide for SmartQR

## Quick Fixes for Common Issues

### üì± **For Mobile Users**

If you're having trouble scanning QR codes on your mobile device, try these solutions:

#### **1. Allow Camera Permissions**
- When prompted, tap **"Allow"** for camera access
- If you accidentally blocked it, look for a camera icon in your browser's address bar
- **Chrome**: Tap the lock icon ‚Üí Camera ‚Üí Allow
- **Safari**: Tap 'aA' ‚Üí Website Settings ‚Üí Camera ‚Üí Allow

#### **2. Use HTTPS Connection**
- Make sure the URL starts with `https://` not `http://`
- Camera access requires a secure connection on mobile devices

#### **3. Refresh After Allowing**
- After granting camera permission, refresh the page
- Pull down to refresh or tap the refresh button

#### **4. Check Browser Compatibility**
- **Recommended**: Chrome, Safari, Firefox
- **Avoid**: In-app browsers (Instagram, Facebook, etc.)
- Update your browser to the latest version

#### **5. Device Settings**
- Go to Phone Settings ‚Üí Privacy ‚Üí Camera
- Ensure your browser has camera permission
- Close other apps using the camera

### üîß **Troubleshooting Steps**

1. **Permission Denied Error**
   - Grant camera access when prompted
   - Check browser settings for camera permissions
   - Refresh the page after allowing access

2. **Camera Not Found**
   - Ensure your device has a working camera
   - Try switching between front/back camera
   - Restart your browser

3. **Camera In Use**
   - Close other apps that might be using the camera
   - Close other browser tabs with camera access
   - Restart your browser if needed

4. **Browser Not Supported**
   - Use Chrome, Safari, or Firefox
   - Avoid using in-app browsers
   - Update to the latest browser version

### üìã **Manual Entry Alternative**

If camera access still doesn't work:
1. Ask your teacher to show you the QR code text
2. Use the "Manual Entry" option in the app
3. Type or paste the code directly

### üÜò **Still Having Issues?**

**For Students:**
- Try using a different device
- Ask a classmate to help scan
- Contact your teacher for assistance

**For Teachers:**
- Ensure students are using HTTPS links
- Provide manual entry codes as backup
- Consider using a different QR code generator

### üí° **Best Practices**

- Always use the latest version of your browser
- Ensure good lighting when scanning
- Hold the camera steady and at the right distance
- Make sure the QR code is clearly visible and not damaged
